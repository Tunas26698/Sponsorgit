Built on Zero Trust Architecture

We know your source code is sacred. We designed SponsorGit to be completely hands-off.

Principle of Least Privilege

We use modern GitHub Apps. No Contents: Read or Write access is ever requested. We only ask for Public Metadata (Stars, Language) and your payout email.

Manual Insert Only

Our servers cannot edit your repository. You manually commit the image tag yourself. Your code remains uncompromised and fully under your control.

External Verification

To verify if the banner is still up, we simply scrape your public GitHub README just like any anonymous guest user would. No backdoor required.

## 🚀 What is SponsorGit?

**SponsorGit** is a sponsorship marketplace that connects **advertisers** (DevTools, SaaS, Cloud companies) with **open-source maintainers** through native banner ads in GitHub READMEs.

> 💡 **One line of Markdown = Recurring revenue.** No tracking pixels. No cookies. No JavaScript injections.

### The Problem
Open-source maintainers power the modern internet, but most earn $0 from their contributions. Traditional ad networks require websites and complex integrations.

### The Solution
SponsorGit lets maintainers add a single Markdown image tag to their README and start earning from the thousands of developers who already visit their repository.

```markdown
[![Sponsored by Vercel](https://api.sponsorgit.com/v1/badge/your-repo)](https://sponsorgit.com/click/your-repo)
```

## 🔧 How It Works {#how-it-works}

```
┌─────────────────────────────────────────────────────────────────┐
│                                                                 │
│   1️⃣  Register    →    2️⃣  Copy Markdown    →    3️⃣  Earn     │
│   Link your repo       Paste one line           Get paid        │
│   on SponsorGit        into README.md           monthly         │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘
```

### For Maintainers
1. **Sign up** and link your GitHub repo
2. **Copy** the generated Markdown badge
3. **Paste** it into your `README.md`
4. **Earn** 95% of the sponsorship revenue every month via Stripe

### For Advertisers
1. **Choose** repositories by topic, language, and star count
2. **Upload** your banner creative
3. **Pay** a flat $19/month per repo slot — no hidden fees
4. **Reach** thousands of qualified developers organically
